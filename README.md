## Quick Deploy (Web UI)
1. Create a new **public** repository (e.g., `accomplishment-cake`).
2. Upload all files from this folder to the repository **root**.
3. Go to **Settings → Pages**.
4. Source: `Deploy from a branch` → Branch: `main` → Folder: `/root` → **Save**.

## Notes
